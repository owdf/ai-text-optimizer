# Changelog

All notable changes to this project are documented here.

## 1.1.0 - 2026-07-17

- Add a default-on local Privacy Shield for credentials, tokens, email addresses, private keys, and Windows user paths.
- Restore protected placeholders only after the provider response returns to the device.
- Add one-click Shorter, Clearer, and Action plan follow-up refinements.
- Show deterministic local change percentage, character delta, and protected-item count in the result window.
- Add bilingual Privacy Shield settings and documentation.
- Expand the regression suite to 72 tests.

## 1.0.1 - 2026-07-17

- Fix the Windows executable build by removing a missing data-file reference.
- Use the portable `AITextOptimizer.exe` executable name.
- Add regression coverage for packaged data files.
- Remove the accidentally committed virtual environment.
- Add Windows CI, packaging smoke tests, and tag-based releases.
- Add a complete MIT license and release-first bilingual documentation.

## 1.0.0 - 2026-06-19

- Initial public version.
